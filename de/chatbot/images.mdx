## Bild

### Verwendung von Markdown

Die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images) ermöglicht es Ihnen, Bilder mit dem folgenden Code hinzuzufügen

```md
![Titel](/pfad/bild.jpg)
```

Beachten Sie, dass die Bilddateigröße weniger als 5 MB betragen muss. Andernfalls empfehlen wir, es bei einem Dienst wie [Cloudinary](https://cloudinary.com/) oder [S3](https://aws.amazon.com/s3/) zu hosten. Sie können dann diese URL verwenden und einbetten.

### Verwendung von Einbettungen

Um mehr Anpassungsmöglichkeiten bei Bildern zu erhalten, können Sie auch [Einbettungen](/writing-content/embed) verwenden, um Bilder hinzuzufügen

```html
<img height="200" src="/pfad/bild.jpg" />
```

### iFrames

Lädt eine andere HTML-Seite innerhalb des Dokuments. Am häufigsten zum Einbetten von Videos verwendet.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
