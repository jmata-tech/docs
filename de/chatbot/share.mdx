## Website-Integration

### Fügen Sie Ihren Chatbot zu Ihrer Website hinzu
Fügen Sie Ihren Chatbot direkt zu Ihrer Website hinzu, indem Sie das folgende Code-Snippet kopieren und in Ihr HTML einfügen. Der Chatbot wird als schwebende Schaltfläche in der unteren rechten Ecke Ihrer Website angezeigt.

```
<script>
    function loadChatScript() {
        var chatID = '7ee6c0e0-c5bb-4b3d-ac82-557ef09b9340';
        var script = document.createElement('script');
        script.src = "https://doko-ai.com/api/chatbot/embed?chatbotId=" + chatID;
        document.body.appendChild(script);
    }
    loadChatScript();
</script>
```

## Web-Oberfläche
Teilen Sie einen direkten Link zur Web-Oberfläche Ihres Chatbots. Benutzer können über ihren Webbrowser auf das vollständige Chat-Erlebnis zugreifen, ohne dass eine Installation erforderlich ist.
Dieser Link wird automatisch generiert, wenn Sie einen Chatbot auf der Plattform erstellen.
```
https://doko-ai.com/chat/chatID
```
