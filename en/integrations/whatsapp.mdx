### Live Mode

Live mode is a useful tool that allows you to transfer the conversation to a human agent. 
When activated, the bot is automatically disabled, allowing the agent to respond directly to customer messages.

Live chat has only 2 states:
- **Enabled:** any message sent to the customer via WhatsApp Business will activate live mode, so bot responses will stop.
- **Disabled:** requires you to manually configure an activation phrase to turn on live chat.

### Reactivation

Set a phrase for users to automatically reactivate the chatbot from live mode. This feature is only available for WhatsApp accounts connected with the mobile application.

The message only needs to contain the configured phrase.

<Tip>
You can configure quick replies in WhatsApp Business to make it easier for users to send the exact phrase. [Learn how to configure quick replies](https://www.facebook.com/business/help/).
</Tip>
